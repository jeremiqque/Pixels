# Pixels

Pixels transforms any image into striking visual art in seconds. Upload a photo, choose from dithering algorithms, geometric shape styles, and stackable post-processing effects — then dial in every detail from palette to pixel size. Create everything from raw halftone prints to glitched-out retro compositions, all inside a fast, precise editor.

## Features

**Dither Mode**
- Floyd-Steinberg, Atkinson, Ordered (Bayer), Burkes, and more
- Threshold and strength controls
- Custom foreground/background colors with transparency support
- Named color palette presets

**Shape Mode**
- Pixel Art, Dot, Lego, Diamond, Lines, Cross, and Blocks styles
- Cell size, min/max fill controls
- Source color or custom palette
- Lines direction (horizontal, vertical, both)

**Post-Effects** — stackable on top of any mode
- Blur, Noise, Vignette, Chromatic Aberration, Glitch
- Each effect independently toggleable with its own parameters

**Editor**
- Crop tool — select and crop the source image before processing
- Export at 1×, 2×, 3×, or 4× resolution
- Low-res preview while dragging sliders for a lag-free experience
- Drag & drop, paste from clipboard, or browse to upload
- Light and dark mode

## Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [React 19](https://react.dev)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
- [Phosphor Icons](https://phosphoricons.com)
- Canvas 2D API for all image processing

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
